# Compliance Notes

This is a beginner-level Streamlit project created for learning and hackathon practice.

## Included

- README
- Contributing guide
- User manual
- Security notes
- Code of conduct
- Changelog
- Environment example
- Docker readiness files
- Basic GitLab CI checks

## Not Included

### Secret Scanning

Not included because the app does not use API keys, passwords, tokens, databases, or external services.

### Dependency Audit

Not included by default. It can be added later using `pip-audit`.

### Test Framework

Not included because the app is intentionally small and mostly UI-driven. Manual testing is documented in the README.

### Coverage Reporting

Not included because no automated test suite exists yet.

## Data Note

The app includes sample finance data for demonstration. It is not real personal financial data.
