# Security Policy

## Reporting Security Issues

This is a beginner Streamlit project and does not handle passwords, payments, bank connections, or real personal finance data.

If you find a security issue, please report it through the repository issue tracker.

## Sensitive Data

Do not commit:

- API keys
- Passwords
- Personal tokens
- Real bank exports
- `.env` files
- Streamlit secrets

## Finance Data Warning

Use anonymized or sample data when deploying publicly. Do not upload private financial records to a public repository.
