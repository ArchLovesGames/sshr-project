# Security Policy

## Reporting Security Issues

This is a beginner Streamlit project and does not handle passwords, payments, or personal user data.
It is very simple and does not use a database or store any data you enter.

If you find a security issue, please report it through the repository issue tracker.

## Sensitive Data

Do not commit:

- API keys
- Passwords
- Personal tokens
- `.env` files
- Streamlit secrets
